In "ONLINE VOTING SYSTEM" a voter and candidate can use his/her voting right online without any difficulty. He/She has to be registered first for him/her to vote or to participate as a candidate. Registration is done by filling the valid details(name,email,phone number,address) in registration page. After successful registration, the voter can able to login . after the login process participated candidate detail will available in dashboard page where voter can able choose and vote for right candidate. these all process will handle by 'ADMIN' where he can add and delete the voter and candidate's data. only admin can able see voting session flow...

NOTE:

  • fill valid details in registration page
  • if registered details were not real,respective voters and candidate will be be removed from database
